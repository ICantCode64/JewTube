## Setting it up
### Dependencies
1. ffmpeg

### Process
To setup jewtube you would need xampp, or any other shit. Oh and you need ffmpeg installed.
1. Download source code
2. Download the database
3. Clean up database if you want <br>
   a) If you did not, please download uploads archive https://archive.org/details/uploads_202605
4. Import db.
5. Done!
